Diferencias entre valores en encuestas

Esta es una implementación interactiva del ejercicio que Gillermo Moncecchi hizo en este cuaderno de iPython basándose en estas notas de Charles Franklin. La idea, en breve, es que cuando se necesita comparar los valores de dos valores reportados en una encuesta (por ejemplo la intención de voto para dos candidatos en una encuesta electoral), para saber si un valor, de acuerdo a la encuesta, es estadísticamente mayor (o menor) que el otro no basta restarlos y comparar ese resultado con el margen de error que reporta la encuesta o algo así pues ese margen de error es para los valores, no para sus diferencias. Las diferencias se distribuyen de forma distinta que los valores puntuales y por tanto tienen otro margen de error que requiere un cálculo mas elaborado. Lo mismo pasa si se comparan dos valores a una misma pregunta en encuestas distintas (por ejemplo, la intención de voto de cierto candidato en dos encuestas y se quiere ver si mejoró o no). Esta pequeña aplicación considera ambos casos y calcula, dados los valores reportados en la encuesta o encuestas y el número de encuestados en las encuestas consideradas, si la diferencia entre los dos valores es estadísticamente significativa con un 95% de confianza. Para más detalles revise el cuaderno de Guillermo o las notas de Franklin.

Instrucciones

Registre los valores como números entre 0 y 1. Si en la encuesta los reportan como porcentajes, divida ese número por 100. En el caso de las dos encuestas, debe registrar el número de encuestados de ambas encuestas. En el caso de solo una encuesta, si la diferencia no es estadísticamente significativa la aplicación le indica cuál sería el mínimo número de encuestados para que una diferencia como la reportada lo fuera.
